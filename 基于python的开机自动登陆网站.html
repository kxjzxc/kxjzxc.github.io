еҹәдәҺpythonзҡ„ејҖжңәиҮӘеҠЁзҷ»йҷҶзҪ‘з«ҷ

еҸ‘иЎЁдәҺ 2019-11-03 жӣҙж–°дәҺ 2019-12-28 еҲҶзұ»дәҺ и®Ўз®—жңә пјҢ зЁӢеәҸ пјҢ Python жң¬ж–Үеӯ—ж•°пјҡ 3.2k йҳ…иҜ»ж—¶й•ҝ ≈ 3 еҲҶй’ҹ

еүҚиЁҖ

иҝ‘ж—ҘдёҺеҗҢеӯҰиҒҠеӨ©пјҢжҲ‘еҒ¶з„¶дәҶи§ЈеҲ°д»–еҶҷдәҶдёҖдёӘиҮӘеҠЁзҷ»йҷҶзҪ‘з«ҷе№¶дё”зӯҫеҲ°зҡ„и„ҡжң¬пјҢй©¬дёҠжңүдәҶе…ҙи¶ЈгҖӮжӯЈеҘҪпјҢжҲ‘еёёеёёеӣ жҜҸж¬ЎејҖжңәйғҪз”Ёзҷ»йҷҶж ЎеӣӯзҪ‘иҖҢж„ҹеҲ°йә»зғҰпјҢзҙўжҖ§е°ұиҮӘе·ұеҶҷдёҖдёӘиғҪеӨҹиҮӘеҠЁзҷ»йҷҶж ЎеӣӯзҪ‘зҡ„и„ҡжң¬гҖӮ

seleniumеә“

Selenium жҳҜдёҖз»„иҪҜд»¶е·Ҙе…·йӣҶ,жҜҸдёҖдёӘйғҪжңүдёҚеҗҢзҡ„ж–№жі•жқҘж”ҜжҢҒжөӢиҜ•иҮӘеҠЁеҢ–гҖӮеӨ§еӨҡж•°дҪҝз”Ё Selenium зҡ„QAе·ҘзЁӢеёҲеҸӘе…іжіЁдёҖдёӨдёӘжңҖиғҪж»Ўи¶ід»–д»¬зҡ„йЎ№зӣ®йңҖжұӮзҡ„е·Ҙе…·дёҠгҖӮз„¶иҖҢпјҢеӯҰд№ жүҖжңүзҡ„е·Ҙе…·дҪ е°ҶжңүжӣҙеӨҡйҖүжӢ©жқҘи§ЈеҶідёҚеҗҢзұ»еһӢзҡ„жөӢиҜ•иҮӘеҠЁеҢ–й—®йўҳгҖӮиҝҷдёҖж•ҙеҘ—е·Ҙе…·е…·еӨҮдё°еҜҢзҡ„жөӢиҜ•еҠҹиғҪпјҢеҫҲеҘҪзҡ„еҘ‘еҗҲдәҶжөӢиҜ•еҗ„з§Қзұ»еһӢзҡ„зҪ‘з«ҷеә”з”Ёзҡ„йңҖиҰҒгҖӮиҝҷдәӣж“ҚдҪңйқһеёёзҒөжҙ»пјҢжңүеӨҡз§ҚйҖүжӢ©жқҘе®ҡдҪҚ UI е…ғзҙ пјҢеҗҢж—¶е°Ҷйў„жңҹзҡ„жөӢиҜ•з»“жһңе’Ңе®һйҷ…зҡ„иЎҢдёәиҝӣиЎҢжҜ”иҫғгҖӮSelenium дёҖдёӘжңҖе…ій”®зҡ„зү№жҖ§жҳҜж”ҜжҢҒеңЁеӨҡжөҸи§ҲеҷЁе№іеҸ°дёҠиҝӣиЎҢжөӢиҜ•гҖӮ

дёӯж–Үж–ҮжЎЈ

е®үиЈ…еә“

жҹҘзңӢеә“

webdriver

Selenium 2.0 жңҖдё»иҰҒзҡ„дёҖдёӘж–°зү№жҖ§е°ұжҳҜйӣҶжҲҗдәҶ WebDriver APIгҖӮWebDriver жҸҗдҫӣжӣҙзІҫз®Җзҡ„зј–зЁӢеҮ еҸЈпјҢд»Ҙи§ЈеҶі Selenium-RC API дёӯзҡ„дёҖдәӣйҷҗеҲ¶гҖӮWebDriver дёәйӮЈдәӣйЎөйқўе…ғзҙ еҸҜд»ҘдёҚйҖҡиҝҮйЎөйқўйҮҚж–°еҠ иҪҪжқҘжӣҙж–°зҡ„еҠЁжҖҒзҪ‘йЎөжҸҗдҫӣдәҶжӣҙеҘҪзҡ„ж”ҜжҢҒгҖӮWebDriver зҡ„зӣ®ж ҮжҳҜжҸҗдҫӣдёҖеҘ—зІҫеҝғи®ҫи®Ўзҡ„йқўеҗ‘еҜ№иұЎзҡ„ API жқҘжӣҙеҘҪзҡ„ж”ҜжҢҒзҺ°д»Јй«ҳзә§ web еә”з”Ёзҡ„жөӢиҜ•е·ҘдҪңгҖӮ

win7+Chrome

жҲ‘зҡ„window7зі»з»ҹдёҠдҪҝз”Ёзҡ„жөҸи§ҲеҷЁжҳҜChromeпјҢжүҖд»ҘжҲ‘йңҖиҰҒдёӢиҪҪдёҖдёӘChormeзҡ„webdriver
йҰ–е…ҲдҪ йңҖиҰҒзҹҘйҒ“зҹҘйҒ“дҪ з”өи„‘дёҠChromeзҡ„зүҲжң¬пјҢдҪ еҸҜд»ҘеңЁChromeзҡ„helpдёӯжүҫеҲ°гҖӮ
з„¶еҗҺиҝӣе…ҘдёӢиҪҪзҡ„зҪ‘з«ҷпјҢдёӢиҪҪеҜ№еә”зүҲжң¬зҡ„webdriverгҖӮ
жҲ‘зҡ„ChromeжҳҜ78зүҲзҡ„пјҢжүҖд»ҘжҲ‘д№ҹдёӢиҪҪдәҶ78зүҲ

жҺҘзқҖе°Ҷе…¶и§ЈеҺӢпјҢе№¶з§»еҠЁеҲ°Chromeзҡ„е®үиЈ…зӣ®еҪ•пјҲжҲ–иҖ…еҲ«зҡ„д»Җд№Ҳзӣ®еҪ•пјүпјҢе№¶дё”ж·»еҠ зҡ„дҪ зҡ„зҺҜеўғеҸҳйҮҸдёӯзҡ„pathдёӯгҖӮ

ubuntu+firefox

жҲ‘зҡ„ubuntuзі»з»ҹдёҠдҪҝз”Ёзҡ„зҒ«зӢҗжөҸи§ҲеҷЁпјҢеӣ жӯӨжҲ‘йңҖиҰҒдёӢиҪҪзҒ«зӢҗзҡ„й©ұеҠЁgeckodriver
еҗҢж ·дҪ йңҖиҰҒзҹҘйҒ“зҹҘйҒ“дҪ з”өи„‘дёҠfirefoxзҡ„зүҲжң¬,з„¶еҗҺиҝӣе…ҘдёӢиҪҪзҡ„зҪ‘з«ҷпјҢдёӢиҪҪеҜ№еә”зүҲжң¬гҖӮ
дёӢиҪҪеҲ°жң¬ең°еҗҺпјҢиҝӣе…ҘжүҖеңЁж–Үд»¶еӨ№пјҢжү“ејҖз»Ҳз«Ҝе№¶жү§иЎҢд»ҘдёӢжҢҮд»Ө

еҲҶеҲ«жҳҜи§ЈеҺӢпјҢиөӢдәҲжқғйҷҗпјҢз§»еҠЁгҖӮ

д»Јз Ғ

ејҖеҸ‘зҺҜеўғжҗӯе»әеҘҪдәҶпјҢжҺҘдёӢжқҘе°ұеҸҜд»Ҙд»Јз ҒдәҶгҖӮ
д»Јз ҒеҫҲз®ҖеҚ•пјҢеҸӘжңүзҹӯзҹӯдәҢеҚҒеҮ иЎҢгҖӮ
йҷӨдәҶиҮӘеҠЁзҷ»йҷҶж ЎеӣӯзҪ‘д№ӢеӨ–пјҢжҲ‘иҝҳеўһеҠ дәҶиҮӘеҠЁжЈҖжөӢзҪ‘з»ңзҡ„еҠҹиғҪгҖӮ
еҰӮжһңзҪ‘з»ңе°ҡжңӘиҝһжҺҘпјҢйӮЈд№ҲзЁӢеәҸе°ҶдјҡдёҚж–ӯең°зҷ»йҷҶж ЎеӣӯзҪ‘пјҢзӣҙеҲ°жЈҖжөӢеҲ°зҪ‘з»ңгҖӮ
жЈҖжөӢзҪ‘з»ңжҲ‘з”Ёзҡ„жҳҜжңҖз®ҖеҚ•зҡ„pingгҖӮ

и®ҫзҪ®ејҖжңәиҮӘеҗҜ

и„ҡжң¬еҶҷе®ҢдәҶпјҢжҲ‘д»¬йңҖиҰҒи®©е®ғдёҖејҖжңәе°ұиҮӘеҠЁиҝҗиЎҢ

windows7

windowsдёҠзҡ„ж“ҚдҪңеҫҲз®ҖеҚ•гҖӮ
йҰ–е…Ҳжү“ејҖеҗҜеҠЁж–Үд»¶еӨ№пјҲејҖе§Ӣ->жүҖжңүзЁӢеәҸ->еҗҜеҠЁ->еҸій”®жү“ејҖпјү,з„¶еҗҺиҮӘе·ұж–°е»әдёҖдёӘbatж–Үд»¶пјҢе№¶д»Ҙи®°дәӢжң¬зҡ„ж–№ејҸжү“ејҖгҖӮ
еңЁbatж–Үд»¶дёӯж·»еҠ д»ҘдёӢиҜӯеҸҘпјҡ

дҝқеӯҳе…ій—ӯпјҢе№¶дё”йҮҚеҗҜжөӢиҜ•пјҢе®ҢжҲҗгҖӮ

ubuntu

ubuntuдёҠеҸҜиғҪжҜ”иҫғйә»зғҰпјҢжҲ‘е°қиҜ•дәҶдёӨз§Қж–№жі•пјҢдҪҶеҸӘе®һзҺ°дәҶдёҖз§ҚгҖӮ

ж–№жі•дёҖ

з¬¬дёҖдёӘж–№жі•жҳҜйҖҡиҝҮи®ҫзҪ®rc.localејҖжңәеҗҜеҠЁе‘Ҫд»Ө/и„ҡжң¬жқҘе®һзҺ°
иҝҷдёҖзҜҮеҚҡж–Үи®Іеҫ—жҜ”иҫғиҜҰз»ҶпјҢжҲ‘иҝҷйҮҢе°ұз®ҖеҚ•и®ІдёҖдёӢж“ҚдҪң

  1. зј–иҫ‘д»ҘдёӢи„ҡжң¬ еңЁжң«е°ҫеҠ дёҠд»ҘдёӢеҶ…е®№
  2. еҲӣе»әе№¶жү“ејҖrc.local еңЁж–Үд»¶дёӯеҶҷе…ҘжҢҮд»Ө
  3. еҲӣе»әиҪҜй“ҫжҺҘ
  4. йҮҚеҗҜжөӢиҜ•

дёҚиҝҮд»ҘеҗҺиҝҳжңүи„ҡжң¬йңҖиҰҒејҖжңәиҮӘеҗҜзҡ„иҜқпјҢеә”иҜҘеҸӘз”Ёзј–иҫ‘rc.localж–Үд»¶еҚіеҸҜгҖӮ
дҪҶжҳҜеҫҲеҸҜжғңиҝҷз§Қж–№жі•жҲ‘е№¶жңӘе®һзҺ°пјҢеҺҹеӣ жҲ‘д№ҹдёҚжҳҜеҫҲжё…жҘҡгҖӮ

з¬¬дәҢдёӘж–№жі•еҲҷжҳҜйҖҡиҝҮubuntuиҮӘеёҰзҡ„ејҖжңәиҮӘеҗҜеә”з”ЁвҖңStartup ApplicationвҖқжқҘе®һзҺ°
